# scripts
Представлены самостоятельно написанные скрипты, из последнего.

auth.php - расширение скрипта https://www.gurock.com/testrail/docs/integrate/auth/ldap для аутентификации пользователей. В исходном скрипте нет функциональности проверки того в какой группе состоит пользователь. На уровне open LDAP нет поддержки memberof. В скрипт добавлена функциональность вычитывания пользователей из одной из ldap групп и проверка входимости текущего пользователя в состав этой группы. Если пользователя в группе нет - кидаем exception и доступа пользователь не получает.

/lua/* - Каталог с результатами  экспериментов по изучению скриптового языка lua.

В целом чтобы разобраться в чужом скрипте на любом языке нужно от нескольких минут до нескольких десятков минут.
